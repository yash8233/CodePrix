# рџљЂ Getting Started

## 1. Clone Repository

```bash
git clone https://github.com/yash8233/CodePrix.git

cd CodePrix
```

---

## 2. Install Dependencies

```bash
npm install
```

---

## 3. Generate Template Files

```bash
npm run generate-templates
```

---

## 4. Configure Environment Variables

Create a `.env.local`

```env
DATABASE_URL=

AUTH_SECRET=

AUTH_GITHUB_ID=
AUTH_GITHUB_SECRET=

AUTH_GOOGLE_ID=
AUTH_GOOGLE_SECRET=
```

---

## 5. Generate Prisma Client

```bash
npx prisma generate
```

---

## 6. Run Ollama

Install Ollama

https://ollama.com

Pull a model

```bash
ollama pull codellama
```

Run it

```bash
ollama run codellama
```

---

## 7. Start Development Server

```bash
npm run dev
```

Application

```
http://localhost:3000
```
